```{myst:static}

```


## "You should notice..." directive

``````{myst:static}
:::{youShouldNotice}
...the `{youShouldNotice}` directive fully supports markdown in the body.
This means you can, for example include some code:

```python
print("Some code!")
```
:::
``````
